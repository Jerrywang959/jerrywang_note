# 为什么用gitbook

1. 记笔记的目的本身就是在有需要时检索，放在公网上易于访问，也有可能帮助到有需要的人。
2. 通过gitbook撰写的非数据库markdown文档，更加具有掌控感。
3. web应用易于扩展和移动，且服务器在国外可以无备案使用域名。
